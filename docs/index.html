SeqDelta

DNA mutation analysis + protein impact interpretation

SeqDelta compares a reference DNA sequence and a mutant DNA sequence, performs explicit alignment, detects substitutions, insertions, and deletions, then separates nucleotide-level events from codon and protein consequences in a presentation-grade report.

What it does

Mutation analysis for real FASTA inputs

  • Parses arbitrary FASTA files from NCBI or manual mutation edits
  • Runs explicit global pairwise alignment
  • Detects substitution, insertion, and deletion events
  • Classifies silent, missense, nonsense, and frameshift effects
  • Exports HTML, JSON, and CSV outputs

How it works

Simple pipeline

1
Load reference and mutant FASTA files.
2
Run global pairwise alignment across the DNA sequences.
3
Detect nucleotide-level mutation events.
4
Translate codons and interpret protein-level consequences.
5
Render a browser-grade report for screenshots, demos, or class presentations.

CLI usage

Install and run in one line

pip install git+https://github.com/milord-x/SeqDelta.git
seqdelta compare reference.fasta mutant.fasta --html-report report.html

After installation, the primary CLI is seqdelta. The repo still keeps python cli.py as a fallback for local development.

Web interface

Local FastAPI app

python app.py

Launch the local browser interface to upload reference and mutant FASTA files and generate the same analysis through a visual scientific dashboard.
